# お父さんのホルマリン漬け風ギミックまとめ

> Last Update: 2026-07-05

## ギミック概要

去年購入した麦茶容器を活用し、透明な容器の中に水を張って、そこにお父さんの顔写真や腕・手などのパーツ写真を入れる。

見た目としては、**人体の一部がホルマリン漬けにされているように見える展示型ギミック** を目指す。

去年は暗くて目立たなかったため、今年は必ず光源を確保する。

## 前提

| 項目         | 内容                                                     |
|--------------|----------------------------------------------------------|
| 使用する容器 | 去年購入した麦茶容器                                     |
| 中身         | 水、顔写真のカラーコピー、腕・手などのパーツ             |
| 容器の条件   | 必ず割れにくいものを使う                                 |
| 水の見た目   | 透明な水より、少し色を付けると雰囲気がそうなので改善する |

去年の反省:

- 光が足りず、容器の中身が見えにくかった
- 装飾がなかった（ストーリー感がなかった）

## 基本構成

| 要素     | 内容                                           |
|----------|------------------------------------------------|
| 容器     | 麦茶容器                                       |
| 液体     | 色付きの水                                     |
| 中身     | お父さんの顔写真のカラーコピー、腕、手（備品） |
| 照明     | 後ろから照らす                                 |
| 装飾     | ラベル、古い紙、番号、注意書き                 |

## 中に入れるもの

### メイン容器

一番目立つ容器には、**お父さんの顔のカラーコピー** を入れる。

#### 撮影時の注意

* 無表情
* 目を見開いた顔
* 少し下から撮った顔
* 口を半開きにした顔
* 顔だけを大きく切り抜いたもの

### サブ容器

いくつかの容器には、顔以外のパーツを入れる。

例:

| 容器  | 中身の案                         |
|-------|----------------------------------|
| 容器A | お父さんの顔                     |
| 容器B | 腕だけ                           |
| 容器C | 不気味な何か（要検討）           |
| 容器D | 何も入っていないがラベルだけある |

## 写真の入れ方

去年と同様に透明な袋に入れて保護する。

## 水の色

去年の反省を踏まえ、単なる透明な水よりも、少し色を付ける。

### 色（検討中）

| 色       | 印象                                           |
|----------|------------------------------------------------|
| 薄い黄緑 | ホルマリン風、標本っぽいのはこれか（第一候補） |
| 薄い黄色 | 古い保存液っぽい（第二候補）                   |
| 薄い茶色 | 劣化した液体、研究室っぽい（第三候補）         |

## 光源

去年の反省点は、暗くて目立たなかったこと。
今年は必ず光源を用意する。

## 設置場所

ブロック2の入口か、出口（**要検討**）。
このギミックは、通路の途中に置くよりも、少し立ち止まって見せる場所配置したい。

## 見せ方の工夫

### ラベルを貼る

容器にラベルを貼って、標本感を出す。

#### ラベル例

* `SAMPLE-01`
* `HEAD`
* `RIGHT ARM`
* `DO NOT OPEN`
* `保存液交換済`
* `検体 No.03`
* `開封厳禁`
* `異常反応あり`

> ラベルは気づいてくれたらいいな程度なので、漢字や英語でもよいかな、と。

## 容器について

容器は必ず割れにくいものを使う。

### 注意点

* ガラス容器は使わない
* 割れにくいプラスチック容器を使う
* 倒れても水がこぼれにくい位置に置く
* 容器のフタは閉める

水が入った容器は重くなるため、落下や転倒に注意する。

## 追加案

### 1. 空の容器を置く

中身がない容器に、`HEAD` や `FATHER` のラベルだけ貼る。

### 2. 容器の近くにメモを置く

例:

```text id="woxbcl"
検体 No.01
反応あり。
夜間に目線が動くため、観察継続。
```

## 必要なもの

| 品目                       | 状態             |

| 麦茶容器                   | 去年の備品を使用 |
| お父さんの顔写真           | カラーコピーする |
